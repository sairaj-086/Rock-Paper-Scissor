# ✊✋✌️ Rock Paper Scissors

A fun browser-based Rock Paper Scissors game built using HTML, CSS, and JavaScript.

## 🚀 Features

- 🤖 Play against computer
- 🎲 Random computer choices
- 🏆 Winner announcement
- 📊 Score tracking
