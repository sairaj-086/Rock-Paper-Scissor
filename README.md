## 🎯 How to Play

1. Click on **Rock**, **Paper**, or **Scissors**.
2. The computer randomly selects its move.
3. The winner of each round is determined using the classic game rules.
4. The first player to reach **5 points** wins the match.
5. Use the **Restart Game** button to start a new game.
6. Switch between themes using the **Theme Buttons** at the top.
